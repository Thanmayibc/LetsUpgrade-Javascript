Console object Methods

There are various console object methods in javascript

Some of them are discussed below :

  1. log(): Output is printed in the console.
  2. warn(): Shows a warning sign in the console.
  3. error(): Shows an error sign in the console.
  4. time(),timeEnd(): Calculates the time requried for an operation to be executed and prints in the console.
  5. count(): Calculates number of time this function is called and prints in the console.
  6. group(),groupEnd(): Creates and ends a group in the console. Within the group , messages can be printed.
  7. clear(): It is used to clear the console.
