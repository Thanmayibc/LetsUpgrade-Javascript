Different ways to declare a variable

1. Declaring variable using 'var'

Here , variables can be globally or locally scoped.

If variables are re-declared inside the block , then variable is changed outside the block also .

2. Using 'let'

Here, variables inside the block can be re-declared , but this will not affect the variable outside the block.

Both let and var are globally scoped

3. Using 'const'

Here, variables cannot be re-declared . It behaves similar to 'let' when it is inside a block.

const variables should be assigned when declared.
