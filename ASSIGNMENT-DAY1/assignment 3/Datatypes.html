Datatypes which are available in Javascript with Examples

1. Number

Number includes Integers and Floating points.

2. String

String represents textual date.

Any elements inside double/single quotes is a string.A lot of operations(Concatenation,Length,Substring) can be performed

3. Boolean

Boolean is a logical datatype and has only two values : true and false.

4. BigInt

BigInt represents Integers which arbitary length (or large Integers).

5. NULL

NULL represents variable whose value is unknown(or empty,nothing).

6. Undefined

Undefined says that no value is assigned to a variable.

7. Object

Objects are collection of properties. They contain different datatypes and methods.
